__interrupt void TIMER1_A1_ISR ( void  )

TIMER_A1 ISR counts down timeouts in the system.

The timer catches several timeouts.

  1. Close Valves Timeout This timeout is set when command to close all valves received and the timeout is being expired until acknoledgement of succesful command execution gotten.
  2. Open Valves Timeout This timeout is set when command to open all valves received and the timeout is being expired until acknoledgement of succesful command execution gotten.
  3. OK Timeout This timeout is reset each time WaterLeak controller riches us. If timeout eventually expired that's concerned as the WaterLeak controller lost.
